# simple-interest-calculator
# Simple Interest Calculator (Máy tính Lãi suất đơn)

Ứng dụng đơn giản giúp tính toán tiền lãi suất đơn dựa trên tiền gốc, lãi suất hàng năm và khoảng thời gian gửi.

## Công thức
Công thức tính lãi suất đơn được áp dụng:
$$A = P \times (1 + r \times t)$$

Hoặc tính tiền lãi riêng:
$$I = P \times r \times t$$

Trong đó:
* **P** (Principal): Tiền gốc ban đầu
* **r** (Rate): Lãi suất hàng năm (dạng thập phân)
* **t** (Time): Thời gian (tính theo năm)
* **I / A**: Tiền lãi / Tổng số tiền thu được

## Hướng dẫn sử dụng
1. Nhập số tiền gốc ($P$).
2. Nhập mức lãi suất ($r$).
3. Nhập khoảng thời gian ($t$).
4. Nhận kết quả tiền lãi tính toán.
